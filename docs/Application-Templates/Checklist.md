# Checklist

- [ ] Componentes existentes foram reutilizados sem alteração de API.
- [ ] Existe apenas uma ação primária clara por contexto.
- [ ] Loading, vazio, erro e sucesso estão previstos.
- [ ] Desktop, notebook, tablet e 390px foram verificados.
- [ ] A página não possui overflow horizontal.
- [ ] Tema claro e escuro preservam contraste.
- [ ] Fluxos completos funcionam por teclado.
- [ ] Foco retorna após fechar Drawer, diálogo ou overlay.
- [ ] Tabelas, árvores, abas, gráficos e feeds possuem nomes acessíveis.
- [ ] Títulos seguem hierarquia lógica.
- [ ] Animações respeitam `prefers-reduced-motion`.
- [ ] Persistência local não substitui persistência de domínio no servidor.
- [ ] Console está livre de erros e warnings.
