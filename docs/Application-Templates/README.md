# Application Templates

Referências completas de composição do Orizon UI para aplicações ASP.NET Core MVC/Razor. Os templates não introduzem componentes públicos: combinam Dashboard, Data, Advanced Forms e Productivity Components existentes.

| Template | Rota |
|---|---|
| Admin Dashboard | `/components/templates/admin` |
| CRUD Workspace | `/components/templates/crud` |
| Entity Form | `/components/templates/forms` |
| Master Detail | `/components/templates/master-detail` |
| Kanban Workspace | `/components/templates/kanban` |
| Analytics Dashboard | `/components/templates/analytics` |
| Settings Workspace | `/components/templates/settings` |
| Authentication | `/components/templates/auth` |
| User Profile | `/components/templates/profile` |
| Errors & Empty States | `/components/templates/errors` |

Veja [CompositionGuide.md](CompositionGuide.md) para adaptar os exemplos e [Checklist.md](Checklist.md) antes de publicar.
