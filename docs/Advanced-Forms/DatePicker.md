# DatePicker

Data acessível usando o calendário nativo, locale do documento e API `min`, `max`, `clear`.
```razor
<orizon-date-picker asp-for="StartDate" label="Data inicial" min="2026-01-01" />
```
O controle nativo oferece teclado e datas localizadas sem biblioteca; datas desabilitadas complexas devem ser validadas no servidor.
