# Acessibilidade

- SVGs possuem papel e rótulo por meio do componente.
- Pontos, barras e fatias são focáveis e associados ao tooltip.
- Legendas usam botões com `aria-pressed`.
- ProgressRing e Gauge emitem valores ARIA.
- Estados loading/error usam `aria-busy`, texto oculto ou `role=alert`.
- Animações respeitam `prefers-reduced-motion`.

Para análises críticas, ofereça também dados tabulares ou resumo textual.
