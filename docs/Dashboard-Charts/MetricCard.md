# MetricCard

Card de métrica com `title`, `value`, `previous`, `variation`, `trend`, `icon`, `sparkline`, `footer`, `helper`, `loading`, `compact`.
```razor
<orizon-metric-card title="MRR" value="R$ 184 mil" variation="+14%" trend="positive" sparkline="[18,24,31]" />
```
A tendência inclui texto acessível e não depende somente de cor.
