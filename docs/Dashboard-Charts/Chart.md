# Chart

Gráfico SVG nos tipos `line`, `bar`, `horizontal-bar`, `area`, `pie`, `doughnut`. API: `title`, `subtitle`, `labels`, `series`, `height`, `legend`, `tooltip`, `colors`, `grid`, `axis`, `loading`, `empty`, `error`, `animate`, `responsive`.
```razor
<orizon-chart type="line" labels='["Jan","Fev"]' series='[{"name":"Vendas","data":[120,150]}]' />
```
Forneça JSON válido e rótulos concisos. Pontos/barras são focáveis; dados essenciais também devem existir em tabela ou texto quando necessário.
