# Fluxo e Preview

Clique -> `OrizonAppearance.set` -> atributos no `html` -> tokens recalculados -> interface e preview atualizados -> LocalStorage -> debounce -> API. Nenhuma navegacao ou recarga ocorre.

O preview usa os mesmos tokens da aplicacao, portanto representa sidebar, topbar, botoes, cards e tabelas com fidelidade.
