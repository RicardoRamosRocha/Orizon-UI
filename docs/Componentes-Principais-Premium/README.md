# Componentes Principais Premium

Sprint fechada que amplia a API de Tag Helpers do Orizon UI com Button, Icon, IconButton, Chip, Switch, Accordion, Tooltip, Skeleton e StatCard. Os componentes usam os tokens e temas existentes e são demonstrados em `/components/premium`.

Importe `orizon.css` e o módulo `orizon.js` por `_content/Orizon.UI`, como no layout administrativo. As APIs existentes do `orizon-button` continuam válidas.
