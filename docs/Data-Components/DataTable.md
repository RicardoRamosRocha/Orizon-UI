# DataTable

Tabela premium sem virtualização. API: `label`, `striped`, `hover`, `pageable`, `sortable`, `searchable`, `selectable`, `compact`, `responsive`, `sticky-header`, `density`, `loading`, `empty`, `empty-title`, `page`, `total-pages` e conteúdo semântico `thead`/`tbody`.

```razor
<orizon-data-table striped hover sortable searchable pageable><thead>...</thead><tbody>...</tbody></orizon-data-table>
```

Inclui região rolável acessível, busca, skeleton, vazio e paginação visual. Ordenação, paginação remota e seleção de negócio são integradas pela aplicação.
