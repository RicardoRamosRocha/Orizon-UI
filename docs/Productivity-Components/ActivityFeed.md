# ActivityFeed

Feed semântico de atividades. Aceita `label`, `grouped` e `compact`.

Agrupe conteúdo com `data-activity-group`, use `data-activity-item` para cada evento e `data-activity-avatar` para a identidade. Datas devem usar `<time>`. O conteúdo permanece livre para nomes, ícones, descrições e metadados do domínio.
