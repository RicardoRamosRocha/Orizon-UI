# TreeGrid

Tabela hierárquica com colunas e seleção. Aceita `label`, `selectable`, `striped` e `compact`.

Linhas pai usam `data-node-id`; descendentes usam `data-parent` com o mesmo valor. Um botão `data-treegrid-toggle` alterna os filhos. Clique, `Enter` e espaço selecionam linhas quando `selectable` está ativo; setas percorrem as linhas visíveis.
