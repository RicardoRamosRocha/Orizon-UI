# Productivity Components

Componentes premium para fluxos corporativos densos, implementados em Tag Helpers, CSS e JavaScript ES6 puros.

| Componente | Uso principal |
|---|---|
| CommandBar | Contexto, busca e ações da página |
| CommandPalette | Busca global de comandos com `Ctrl+K` |
| Drawer | Painel lateral ou contextual |
| SplitView | Dois painéis redimensionáveis |
| TreeView / TreeGrid | Dados hierárquicos |
| Timeline | Sequência de eventos |
| Kanban | Fluxo visual com drag-and-drop |
| ActivityFeed | Registro de atividades |
| QuickActions | Atalhos favoritos |

Os estilos são incluídos por `orizon.css` e os comportamentos por `orizon.js`. Veja a vitrine em `/components/productivity`.
