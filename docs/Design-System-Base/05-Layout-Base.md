# Layout Base

## Objetivo

Definir a estrutura visual padrão utilizada pelas aplicações da plataforma Orizon.

O layout deverá ser moderno, responsivo, reutilizável e independente da regra de negócio de cada produto.

---

# Estrutura Geral

O layout administrativo será composto pelos seguintes elementos:

* Sidebar
* Topbar
* Área de Conteúdo
* Footer
* Overlay para dispositivos móveis

---

# Sidebar

A Sidebar será o principal elemento de navegação.

Características:

* Largura fixa em desktop
* Recolhível (Collapsed)
* Overlay em dispositivos móveis
* Suporte a múltiplos níveis de menu
* Ícones em todos os itens
* Destaque para item ativo

---

# Topbar

A barra superior conterá:

* Botão para abrir/recolher a Sidebar
* Campo de pesquisa (opcional)
* Ações rápidas
* Notificações
* Perfil do usuário
* Menu do usuário

---

# Área de Conteúdo

Responsável por exibir as páginas da aplicação.

Deverá possuir:

* Espaçamento padronizado
* Largura fluida
* Grid responsivo
* Breadcrumb
* Cabeçalho da página
* Área para ações principais

---

# Footer

O rodapé será simples e reutilizável.

Poderá conter:

* Nome do produto
* Versão
* Direitos autorais
* Links institucionais

---

# Responsividade

## Desktop

* Sidebar fixa
* Conteúdo expandido

---

## Tablet

* Sidebar recolhida por padrão
* Navegação otimizada

---

## Mobile

* Sidebar em modo Overlay
* Conteúdo em largura total
* Navegação por toque

---

# Objetivos

O Layout Base deverá oferecer:

* Navegação intuitiva
* Excelente aproveitamento de espaço
* Alta produtividade
* Consistência visual
* Facilidade de reutilização entre todos os produtos Orizon.
