# Design Tokens

Os Design Tokens representam os valores fundamentais utilizados em toda a interface.

## Tokens previstos

### Cores

* Primary
* Secondary
* Success
* Warning
* Danger
* Info
* Neutral
* Background
* Surface

---

### Tipografia

* Font Family
* Font Size
* Font Weight
* Line Height
* Letter Spacing

---

### Espaçamentos

Escala baseada em múltiplos de 4px.

| Token | Valor |
| ----- | ----: |
| xs    |   4px |
| sm    |   8px |
| md    |  16px |
| lg    |  24px |
| xl    |  32px |
| xxl   |  48px |

### Bordas

### Z-Index

### Duração das animações

Todos os componentes do Orizon UI deverão utilizar exclusivamente estes tokens, evitando valores fixos sempre que possível.
