# Tipografia

## Objetivo

Definir um sistema tipográfico consistente para toda a plataforma Orizon.

A tipografia deverá proporcionar excelente legibilidade, hierarquia visual e aparência profissional.

---

# Princípios

* Escala consistente
* Boa leitura em desktop e mobile
* Hierarquia clara
* Espaçamento equilibrado
* Peso adequado para cada contexto

---

# Estrutura

## Display

Utilizado em:

* Landing Pages
* Hero Sections
* Grandes destaques

## Body Large

Texto principal com maior destaque.

---

## Body

Texto padrão da aplicação.

---

## Body Small

Informações secundárias.

---

## Caption

Utilizado para:

* Legendas
* Auxílio ao usuário
* Informações complementares

---

## Label

Utilizado em:

* Campos de formulário
* Componentes
* Botões
* Navegação

---

## Font Weight

A biblioteca deverá padronizar os seguintes pesos:

* Light
* Regular
* Medium
* SemiBold
* Bold

---

## Line Height

Cada nível tipográfico possuirá altura de linha adequada para maximizar a leitura.

---

## Responsividade

A escala tipográfica deverá adaptar-se automaticamente entre:

* Mobile
* Tablet
* Desktop
* Monitores de alta resolução

---

## Diretrizes

Toda a interface deverá utilizar exclusivamente os estilos tipográficos definidos pelo Design System, evitando tamanhos, pesos ou espaçamentos arbitrários para manter consistência visual em todos os produtos da plataforma.
