# Utilitários CSS

## Objetivo

Disponibilizar classes utilitárias reutilizáveis para acelerar o desenvolvimento e reduzir CSS duplicado nas aplicações.

---

# Categorias

## Display

* Flex
* Grid
* Block
* Inline
* None

---

## Flexbox

* Direction
* Wrap
* Justify Content
* Align Items
* Align Self
* Gap

---

## Espaçamentos

Classes para:

* Margin
* Padding

Escalas baseadas nos Design Tokens.

---

## Dimensões

* Width
* Height
* Max Width
* Min Width

---

## Bordas

* Border
* Border Radius
* Border Color

---

## Cores

Utilitários para:

* Background
* Text
* Border

Sempre utilizando os Design Tokens.

---

## Tipografia

Classes para:

* Font Weight
* Font Size
* Text Align
* Text Transform
* Truncate
* Line Clamp

---

## Sombras

* Shadow XS
* Shadow SM
* Shadow MD
* Shadow LG
* Shadow XL

---

## Visibilidade

* Hidden
* Visible
* Responsive Visibility

---

## Helpers

Também serão disponibilizados utilitários para:

* Cursor
* Overflow
* Position
* Opacity
* Z-Index
* Transitions

---

# Objetivo

Os utilitários deverão complementar os componentes do Design System, mantendo consistência visual e evitando estilos específicos repetidos em cada aplicação.
